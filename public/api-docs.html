HeatBlock API

REST API for getting Minecraft server status information for both Java and Bedrock editions

🚀 Quick Start

Java Edition

GET Request:

/api/status/hypixel.net

Bedrock Edition

GET Request:

/api/status/bedrock/play.cubecraft.net
GET /api/status/{serverAddress}

Get status information for a Minecraft Java Edition server.

Parameters

serverAddress string required

Server hostname or IP address. Can include port (e.g., "example.com:25565"). Default port is 25565.

Example Response

{
  "success": true,
  "version": {
    "name": "1.20.1",
    "protocol": 763
  },
  "players": {
    "max": 100000,
    "online": 45678,
    "list": []
  },
  "description": "§aHypixel Network §c[1.8-1.20]",
  "description_clean": "Hypixel Network [1.8-1.20]",
  "latency": 45,
  "favicon": "data:image/png;base64,..."
}

Try it out

GET /api/status/bedrock/{serverAddress}

Get status information for a Minecraft Bedrock Edition server.

Parameters

serverAddress string required

Server hostname or IP address. Can include port (e.g., "example.com:19132"). Default port is 19132.

Example Response

{
  "success": true,
  "motd": "CubeCraft Games",
  "motd_clean": "CubeCraft Games",
  "levelName": "world",
  "playersOnline": 15234,
  "playersMax": 50000,
  "gamemode": "Survival",
  "serverId": "123456789",
  "protocol": 594,
  "version": "1.20.10",
  "latency": 67
}

Try it out

GET /api/png/{serverAddress}

Get the server favicon as a PNG image (Java Edition only).

Parameters

serverAddress string required

Server hostname or IP address with optional port.

Example

/api/png/hypixel.net

Returns a PNG image or JSON error if no favicon is available.

❌ Error Responses

Common Error Format

{
  "success": false,
  "error": {
    "code": "timeout",
    "message": "Connection to server timed out"
  }
}
timeout

Connection to server timed out

invalid_domain

The domain name could not be resolved

connection_refused

Server refused the connection

offline

Server appears to be offline or unreachable
